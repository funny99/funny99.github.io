Hexo Introduction

一 简介

Hexo是一个快速、简洁且高效的博客框架（静态博客生成器）。Hexo使用Markdown解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

二 环境配置

安装Node

安装Git

安装Sublime（可选）

安装markdown编辑器（可选）

三 GitHub

注册账号、新建仓库（name必须和用户名一致，如funny99.github.io）、添加SSH公钥到『Account settings -> SSH Keys -> Add SSH Key』
